Small, well-tested open-source fixes from CodeAustral LLC.

We help maintainers close scoped issues with clear pull requests, focused tests, and quick review follow-up. Most of our work is in TypeScript, React, Node.js, Python, Go, docs, and test infrastructure.

## How We Work

- We read the issue, contribution guide, and existing PRs before touching code.
- We keep changes small and aligned with the project’s style.
- We include verification notes and tests where practical.
- We avoid duplicate work and ask clarifying questions when acceptance criteria are unclear.
- Payment and admin details are handled privately through platform-supported bounty flows.

## Preferred Work

- Bug fixes with a clear reproduction
- UI and product polish with concrete acceptance criteria
- Documentation improvements that unblock users
- Test coverage for known regressions
- Build, CI, and integration fixes

## Featured Work

- [github-bounty-scout](https://github.com/codeaustral-oss/github-bounty-scout): CLI for reviewing GitHub issue signals before starting bounty work.
